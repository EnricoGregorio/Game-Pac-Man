# Game Pac Man
 Jogo Pac-Man feito em Java!

(PT-BR)
# Teclas de comando

* **A** ou **←** para andar para a **esquerda**.

* **D** ou **→** para andar para a **direita**.

* **W** ou **↑** para andar para **cima**.

* **S** ou **↓** para andar para **baixo**.

(EN-US)
# Command keys

* **A** or **←** to walk to the **left**.

* **D** or **→** to walk to the **right**.

* **W** or **↑** to walk **up**.

* **S** or **↓** to move **down**.
